Given two strings `s` and `t`, return `true` if `t` is an
anagram of `s`, and `false` otherwise.



### Example 1:

__Input__: s = "anagram", t = "nagaram"  
__Output__: true

### Example 2:

__Input__: s = "rat", t = "car"  
__Output__: false


Constraints:

    - `1 <= s.length, t.length <= 5 * 104`
    - `s and t consist of lowercase English letters.`
